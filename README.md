# ScikitLearn-Machine-Learning

                Requirements: 
                Python (>= 2.6 or >= 3.3),
                NumPy (>= 1.6.1),
                SciPy (>= 0.9).
                
To install Scikit follow the steps : 

    pip install -U scikit-learn
or : 

    conda install scikit-learn

Documentation  : http://scikit-learn.org/stable/install.html
